# Task Pilot 🚀

Task Pilot is a simple and interactive task management web application built using HTML, CSS, and JavaScript. Users can create projects, add tasks with different priorities, manage tasks efficiently, and delete projects or tasks easily.

## Features ✨

* Create and manage projects
* Add tasks with priority levels
* Delete tasks and projects
* Clean and responsive UI
* Dark mode support
* Fully frontend-based application

## Tech Stack 🛠️

* HTML
* CSS
* JavaScript

## Project Structure 📂

```bash
Task-Pilot/
│
├── index.html
├── style.css
└── script.js
```

## How to Run ▶️

1. Clone the repository

```bash
git clone <your-repository-link>
```

2. Open the project folder

```bash
cd Task-Pilot
```

3. Run `index.html` in your browser
